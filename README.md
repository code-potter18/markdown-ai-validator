# Automated Markdown Validator

This project is an automated validator for Markdown (.md) files.  
It detects issues such as:
- Broken links
- Missing headings
- Empty headings

## Technology Used
- Python
- Flask
- Agno framework

## How to Run the Project
1. Install required libraries  
   pip install -r requirements.txt
2. Run the application  
   python app.py
3. Open browser and go to  
   http://127.0.0.1:5000(there is local running so i will attach the demo video)
