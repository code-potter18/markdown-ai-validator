## Markdown Validator Documentation

This document describes the Markdown Validator project in detail.

The project validates markdown files for common issues and provides
clear suggestions to fix them.

---

## Project Purpose

Markdown files are often used for technical documentation.
Even small mistakes can cause rendering problems.

The validator ensures markdown quality before deployment.

---

## Key Features

- Detects broken hyperlinks
- Checks for correct heading structure
- Ensures headings are not empty
- Works entirely on a local system

---

## Setup Instructions

1. Download the project files.
2. Create and activate a Python virtual environment.
3. Install dependencies using pip.
4. Start the Flask application.
